This plugin will allow you to Add a logo to your login page.

== Description ==
 
This plugin allows you to customize your admin by adding your logo to the header. 

It also replaces the WordPress logo on the login screen with the same custom logo. 

Tested in Firefox, Safari, Chrome and IE.

== Installation ==

1. Unzip the admin-logo-changer.zip file.
2. Upload the `admin-logo-changer` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to Settings => Admin Logo Changer and set your options and add your logo.
